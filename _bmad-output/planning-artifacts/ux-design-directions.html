Design Directions for the Deploy Briefing Experience

These directions all preserve the same UX principles: verdict first, one-screen operational clarity, visible uncertainty, and evidence on demand. What changes is the balance between summary calm, data density, and how strongly the interface leans toward cockpit, review, or observability metaphors.
