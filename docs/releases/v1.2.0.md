# DeployWhisper v1.2.0

DeployWhisper v1.2.0 is the UI migration release line. It keeps the existing
NiceGUI experience available while adding the read-only backend-for-ui contract
needed by the React dashboard.

## Backend-For-UI Additions

- `GET /api/v1/stats/summary` returns dashboard KPI totals and seven-bucket
  sparkline series for total analyses, clean-verdict rate, high/critical count,
  and average time-to-verdict.
- `GET /api/v1/stats/verdict-distribution?days=30` returns advisory
  recommendation counts for the verdict distribution view.
- `GET /api/v1/projects` remains the project list endpoint and now includes
  `name` and `env_label` aliases for the project switcher.
- `GET /api/v1/analyses` keeps its existing response contract and adds
  dashboard aliases for score, verdict, filenames, workspace/environment labels,
  and trigger references.

## Version Metadata

- Runtime API metadata now reports application version `1.2.0` by default.
- `pyproject.toml` now reports version `1.2.0`.
